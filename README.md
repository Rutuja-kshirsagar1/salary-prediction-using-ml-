# salary-prediction-using-ml-

Overview

This project demonstrates how to build and evaluate machine learning models for predicting salaries based on years of experience. It includes implementations of Linear Regression ,allowing comparison between interpretability and predictive accuracy.

Dataset

The dataset contains two main columns:

Experience Years: Number of years of professional experience.

Salary: Annual salary corresponding to the experience.

dataset : https://raw.githubusercontent.com/ybifoundation/Dataset/main/Salary%20Data.csv

Models Used

Linear Regression

Provides a simple, interpretable formula.

Useful for understanding the relationship between experience and salary.

Visualization

The project includes scatter plots of actual data and overlays predictions from both models:

Blue dots: Actual data points.

Red line: Linear Regression predictions.

Green dots: Random Forest predictions.

Future Improvements
